# CMCrawl
## A Cardmarket list creator

### Installation
```bash
$ git clone https://github.com/DrankRock/CMCrawl.git
$ cd CMCrawl
$ python3 -m pip install -r requirements.txt
```

### Run
``python3 CMCrawl.py``
